# Équilibrage des débits d'air pour VMC

## Principe fondamental

Dans une installation VMC (Ventilation Mécanique Contrôlée), l'équilibre entre les débits d'entrée d'air (par les fenêtres) et les débits d'extraction (par les bouches) est essentiel au bon fonctionnement du système.

## Entrées d'air

Les entrées d'air sont généralement installées sur les menuiseries des pièces principales (séjour, chambres) et permettent l'admission d'air neuf dans le logement.

### Types d'entrées d'air

1. **Autoréglables** : Débit fixe quelle que soit la dépression (ex: 15, 22, 30 ou 45 m³/h)
2. **Hygroréglables** : Débit variable en fonction de l'humidité (ex: 6-45 m³/h)

## Débits réglementaires

### Arrêté du 24 mars 1982 modifié

Débit total minimal extrait (et donc aussi admis par les entrées d'air) :

| Type de logement | Débit total minimal (m³/h) |
|------------------|----------------------------|
| T1               | 35                         |
| T2               | 60                         |
| T3               | 75                         |
| T4               | 90                         |
| T5               | 105                        |
| T6               | 120                        |
| T7 et plus       | 135                        |

## Équilibrage air neuf / air extrait

Pour assurer un bon fonctionnement de la VMC :
- Le débit **total des entrées d'air** (modules des fenêtres) doit être équivalent au débit **total d'extraction** des bouches VMC
- Un déséquilibre génère des dysfonctionnements :
  - Trop d'entrées d'air → courants d'air, surconsommation
  - Pas assez d'entrées d'air → difficultés d'ouverture des portes, mauvais renouvellement d'air

## Dimensionnement des entrées d'air

### 1. Cas VMC autoréglable

- **Séjour / salon** : module 45 ou 2x module 22
- **Chambres** : module 30 (ou 2x module 15)

### 2. Cas VMC hygroréglable type A

- **Séjour / salon** : module 45 ou hygroréglable 6-45
- **Chambres** : module 22 ou hygroréglable 6-22

### 3. Cas VMC hygroréglable type B

- Toutes les entrées d'air sont hygroréglables
- **Séjour / salon** : hygroréglable 6-45
- **Chambres** : hygroréglable 6-22

## Vérification de la cohérence

Pour vérifier l'équilibre du système :
1. Faire la somme des modules d'entrées d'air
2. Comparer avec la somme des débits extraits
3. La différence ne doit pas excéder 10% pour un fonctionnement optimal
