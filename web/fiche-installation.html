Fiche pratique installation VMC

Voici les étapes clés pour une installation réussie de votre VMC :

  1. Choisir le type de VMC adapté à votre logement (simple ou double flux).
  2. Installer les bouches d’extraction dans les pièces humides (cuisine, salle de bain, WC).
  3. Veiller à la bonne étanchéité des gaines et à leur isolation.
  4. Positionner le groupe moteur dans un endroit accessible et peu bruyant.
  5. Raccorder les entrées d’air dans les pièces principales.
  6. Effectuer un test de débit pour vérifier la conformité de l’installation.
